Микросервис для вывода списка фильмов, которые хранятся на жестком диске.

1. Вначале необходимо запустить add_list.py <folder>, чтобы добавить в БД список видео файлов из указанной папки. Это действие можно проделать для нескольких папок. 

2. Затем запускается web.py, который делает запросы к kinopoisk.ru и формирует список фильмов с описаниями, рейтингом и длительностью. 

3. Веб-интерфейс доступен по адресу localhost:80.

4. Зависимости: flask, requests, kinopoiskpy
